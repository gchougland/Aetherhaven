# Sprinklers

Sprinklers are metal friends for big fields. Activate one and it refreshes crops in an area aroudn the sprinkler once daily at dawn. It can also be activated manually by using the block. Unlocked by befriending the Farmer. There are four tiers of sprinkler to unlock with increasing radius of watering.
