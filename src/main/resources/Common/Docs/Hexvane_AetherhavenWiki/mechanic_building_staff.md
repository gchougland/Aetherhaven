# Building staff

After a plot sign exists, the building staff lets you grow the structure in stages. Aim at highlighted blocks and confirm each batch.

After pressing BUILD on a a plot sign, the building will start to build itself slowly. Using a Building Staff lets you speed up this process. While holding the staff green cubes will appear where blocks need to be placed. Holding secondary action (Right mouse button) with the staff while looking at these cubes will cause the cubes to expand and place the blocks for the building. If you can't find any green cubes you can use the primary action (Left mouse button) to shoot a projectile that homes in on the closest green cube. There are multiple tiers of building staff with increasing radius of building with the staff.

**Tips**

* Use scaffolding to reach higher places with the staff.
* Very large buildings are split into sections when building for performance.
