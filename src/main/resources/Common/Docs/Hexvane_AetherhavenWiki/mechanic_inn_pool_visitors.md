# Inn pool and visitors

The inn can host visitors who are not full time residents. The inn can hold up to 2 visitors at a time, refreshing and rerolling daily at dawn. If you have accepted a visitors quest they are locked in and won't be rerolled.

## Debugging

If the inn's visitors don't seem to be showing up or are messed up in some way, try running the `/ah villager fixinn` command.
