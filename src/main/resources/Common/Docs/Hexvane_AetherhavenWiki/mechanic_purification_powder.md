# Purification powder

Purification powder allows the player to remove enemy spawn points in generated structures. While holding the powder the player will be able to see small spinning versions of mobs where they would periodically spawn in the structure. Using the powder on this will remove the spawn point permanently. Unlocked by befriending the Priestess.
